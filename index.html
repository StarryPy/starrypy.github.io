StarryPy

Starbound server wrappers written in Python

StarryPy and StarryPy3K are Python-based server wrappers written to extend the functionality of the vanilla Starbound Server. With an extensible plugin system, a long list additional options can be added to servers including planet protections, party chat, IRC bridging, extended warping, messages of the day, and more!
